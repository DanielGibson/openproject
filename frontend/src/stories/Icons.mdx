# Icons

Icons are used as visual indicators or small illustrations.

<Canvas of={IconsStories.Basic} />

It is very much not recommended to use Icons on their own, especially if they are supposed to convey an action taken on
an interactive element. For example, a button that saves changes to a work package with only the `checkmark` icon as
content is a big no-no.

# Usage

Icons are a CSS class only. To use them, just apply the `spot-icon` BEM block on a generic inline element like `span`.
